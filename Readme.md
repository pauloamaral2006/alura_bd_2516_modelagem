# Modelagem de banco de dados relacional

## ℹ️ Sobre

<p>Projeto utilizado no curso para aprendizado das técnicas e melhores práticas.</p>

## 📘Ementa

### Modelagem de banco de dados relacional: modelagem lógica e física

- Aprender a diferençar os tipos de modelagem de banco de dados
- Conhecer a modelagem lógica e física de dados
- Exercitar a utilização do Power Architect
- Criar tabelas, colunas e representações de cardinalidade no modelo físico
- Produzir a representação das chaves primárias e estrangeiras
- Desenvolver todo o processo de modelagem, desde a modelagem de negócio até o final do projeto
